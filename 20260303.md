## Kali mit Github Konfigurieren
- öffne den Shell in VS Code
- Setze deinen GitHub-Namen und deine E-Mail mit: 
> git config --global user.name "DeinName"

und 

>git config --global user.email "deinemail@example.com"

- Name und Email jeweil in das vorgegeben orte einfügen
