### Variablen 
```bash

```
### File & Directory anlegen
```bash
FILE_NAME=test.txtS
DIR_NAME=mydir
mkdir $DIR_NAME

```

### Variablen ausführen 
```bash
TEXT="Hallo Welt"
echo $TEXT
```

### Double quotes
```bash
# Double quotes
NAME=Mate
TEXT="\$Hallo $NAME"
echo $TEXT
```

### Single quotes
```bash 
#Single quotes
TEXT2='Hallo $NAME'
echo $TEXT
```

### Curly braces {}
Falsch:
```bash
# curly braces {}
FILE=myname
echo "$FILE_001.txt"
echo "$FILE_002.txt"
```
Richtig:
```bash 
FILE=myname
echo "${FILE}_001.txt"
echo "${FILE}_002.txt"
```
