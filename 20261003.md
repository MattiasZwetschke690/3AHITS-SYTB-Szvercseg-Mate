## Example.com mit curl rufen

- Terminal öffnen
> curl www.example.com 

eingeben

- öffne einen Webbrowser und suche nach "www.example.com"
- gehe dann rechts oben auf die drei striche dann -> more tools -> webdeveloper tools
- Damit kannst du die HTML code für die Webseite sehen 
- Vergleiche die code im browser mit dem Code im Terminal

## Example.com zusätzlich den HTTP-Response-Header mit curl rufen 

- [curl options](https://www.kali.org/tools/curl/) 
- im Terminal 
> curl -i www.example.com 

eintippen
- mit -i ist der HTTP-Response-Header sichtbar
- 
## Verbose mode
- [curl options](https://www.kali.org/tools/curl/) 
- Terminal öffnen 
>curl -v https://htl-braunau.at/live

- mit -v ist der Request, Response und Header sichtbar sind

eintippen

## Was bedeutet dieser http Status Code (404)? 
- Der Statuscode 404 – Not Found bedeutet, dass der Server die angeforderte Ressource nicht finden kann.

### Was zeigt ein Web-Browser bei Aufruf dieser URL an?
This httpbin.org page can’t be found
No webpage was found for the web address: https://httpbin.org/status/404

HTTP ERROR 404
